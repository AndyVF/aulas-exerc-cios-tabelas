TABELA DE TAGS

TAGS E SUAS FUNÇÕES
TAGS FUNÇÕES
Table O elemento HTML Table representa dados em duas dimensões ou mais.
TH O elemento HTML TH define uma célula cabeçalho do grupo de células de sua tabela
TR TR define uma linha numa tabela.
TD Uma tabela é dividida em linhas (com a tag tr), e cada linha é dividida em células de dados (com a tag td)
Thead O marcador THEAD /THEAD "Table Head Element" ou linha de cabeçalho de uma tabela HTML, é utilizado para destacar os títulos das colunas de uma tabela
Tbody O elemento TBODY estrutura todo o conteúdo da tabela, para que assim o navegador qual exatamente é o conteúdo da tabela
Tfoot tag TFOOT para combinar conteúdo rodapé da tabela HTML
Align Tem como função centralizar textos e imagens
Border Cada elemento pode ter uma margem interna (padding), margem externa (margin) e uma borda (border).
Caption O Elemento HTML caption (ou Elemento HTML Subtitulo de Tabela) representa o título de uma tabela.
Rowspan Especifica o número de linhas a serem ocupados por uma célula.
Colspan colspan serve para expandir as células fazendo com que uma única célula ocupe mais de uma linha ou mais de uma coluna.4
Width A propriedade CSS width determina a largura da área de conteúdo de um elemento.
